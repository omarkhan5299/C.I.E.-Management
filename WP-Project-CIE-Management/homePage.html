CIE Eligibilty Evaluation

The CIE Evaluation application has been developed to help automate the process of evalutaing the eligibilty of students for writing the Semester End Examination. The appication allows the user to upload an excel sheet which contains information about the marks and attendance of the student. The application extracts data from the uploaded excel file into our database and automates the process of evaluation. The evaluated results are further written into a formatted excel sheet which can be downloaded by the user. Our application aims to reduce dependance on paper for official use and saves time.

The uploaded excel sheet should be in the following synatx:

S.No. | USN | Name | Marks | Attendance

Data should be entered from second row onwards.
